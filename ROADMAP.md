# Roadmap

## MVP Scope

- Mock JSON store data
- Search by city, postal code, or store name
- Service, open now, and distance filters
- Distance-sorted results
- Real OpenStreetMap/Leaflet map
- Mobile sticky filters
- Responsive desktop split layout
- Store detail drawer
- Empty, loading, and error states

## Phase 2 Ideas

- Mapbox integration for brand-styled map tiles
- Browser geolocation with permission education
- Store opening hours by day
- Appointment availability by service
- Analytics event tracking
- Saved favorite store
- Localized French copy option for Paris rollout

## Nice-to-have Features

- “Open this weekend” filter
- Public transport travel time
- Service price preview
- Promotion badges
- Store photo carousel
- Recently viewed locations

## Technical Improvements

- Add unit tests for filtering logic
- Add Playwright visual checks for mobile and desktop
- Instrument analytics events
- Add URL query params for shareable filtered states
- Replace mocked map with Mapbox provider abstraction
